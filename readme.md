# 🔍 TruthLens — Verificador Inteligente de Informações com IA

Sistema web full stack para análise de fake news em texto, imagem e áudio, utilizando inteligência artificial e recurso de denúncia anônima.

---

## 🧱 Estrutura e Organização do projeto

Cada funcionalidade possui a seguinte estrutura e organização de pastas e arquivos:


| /assets/ | função |
|---------------|--------|
| css/ | Reponsável pela estilização das páginas |
| fonts/ | Arquiva as diferentes fontes utilizadas no sistema |
| imagens/ | Arquivo de imagens utilizadas em cada página | 
| js/ | Scripts do JavaScript reponsável pela interação do sistema |

| /pages/ | funcão |
|---------|--------|
| html | Reponsável pela estruturação do conteúdo aprensentado em cada página |

### Outras estruturas
- index.html -> Tela principal do projeto, responsável pela inicialização.
- readme.md -> Responsável pela documentação do projeto
